# PaperboardServer
Interactive paperboard websocket server. 

# Installation 
- clone the repo
- download the maven dependencies written in the pom.xml file
- start the server with run
- setup your jdk to 11.0.4

## Route HTTP
- get pseudo availability
- get all rooms available
- create rooms
- export board

## Socket
- join room
- draw
- edit
- Delete
- move ?
- leave
